BUTONLAR

Özellikler

text => string / Butonun yazısı için kullanılır isteğe bağlıdır. İstendiğinde child olarak yazı eklenebilir.

title => string / Butonuna html title özelliği ekler.

loading => boolean / Botunu disable yapıp yüklenme animasyonu ekler.

disabled => boolean / Botunu disable yapar.

clicked => func / Click eventi callback.
