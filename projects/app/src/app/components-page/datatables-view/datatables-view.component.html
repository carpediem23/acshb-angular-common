Tablolar

Özellikler

columns => Array / Datatable columns verisi Json olarak set edilir. Json özellikleri aşağıda verilmiştir.

name => string / Json scheme daki field ismi.

title => string / Tabloda görüntülenmek istenilen başlık.

sort => boolean/string / Tabloda sıralama özelliği istenilirse verilmelidir. 'asc' veya 'desc' olarak set edilir. İstenmiyorsa false olarak set edilmelidir.

filtering => object / Tabloda ilgili alanda arama yapılma istenilirse set edilmelidir. Yanlızca 'placeholder' isminde alan ile set edilmelidir.
